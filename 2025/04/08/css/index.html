BFC

块级格式化上下文（Block Formatting Context），它是浏览器在渲染的过程中用于组织块级元素的一个独立渲染区域

触发 BFC 方式

  • float 的值不是 none。
  • position 的值不是 static 或者 relative。
  • display 的值是 inline-block、table-cell、flex、table-caption 或者inline-flex
  • overflow 的值不是 visible

为了解决：

  1. 外边距合并
  2. 浮动元素的影响
  3. 文字环绕效果
  4. 防止外边距穿透
